# 화상 및 출혈 응급처치 가이드
출처: 행정안전부, 안전교육교안, 보건소 응급처치 가이드

## 화상 응급처치

### 화상이란?
불, 뜨거운 증기, 기름, 물, 주방기구 등에 의해 피부가 손상되는 것으로 주로 어린이에게서 많이 발생

### 화상의 정도 분류

#### 1도 화상 (경미한 화상)
- 표피만 손상
- 피부가 빨갛게 변하고 통증이 있음
- 물집이 생기지 않음
- 2~3일 내 자연 치유
- 흉터가 남지 않음

#### 2도 화상 (중등도 화상)
- 진피까지 손상
- 물집이 생김
- 심한 통증
- 물집을 터뜨리지 않으면 1~2주 내 치유
- 얕은 흉터가 남을 수 있음

#### 3도 화상 (심한 화상)
- 피부 전층 손상
- 피부가 하얗거나 검게 변함
- 신경이 손상되어 통증을 못 느낄 수 있음
- 피부 이식이 필요할 수 있음
- 심한 흉터가 남음
- **즉시 응급실 방문 필요**

### 화상 응급처치 방법

#### 1단계: 화기 제거 및 냉각 (가장 중요!)

**미온수로 10~15분간 세척**
- 화상 부위를 **미온수의 깨끗한 물이나 식염수로 10~15분간** 세척
- 흐르는 찬물 속에 넣어 적어도 10분 동안 담금
- 후끈거리는 열감을 빼주지 않으면 피부 속으로 깊이 침투하여 손상이 심해짐
- **즉시 화기를 식혀주어야 함**

**주의사항**
- **얼음을 직접 상처 부위에 대지 않음** - 동상 위험
- 어린이나 노약자의 경우 차가운 물로 장시간 식혀주거나 몸 전체를 담가놓으면 떨림 증상이 올 수 있음
- 이 경우 잠시 중지하고 화상 부위를 제외한 곳을 보온

#### 2단계: 의복 처리

**올바른 방법**
- 화상 부위에 밀착된 의복을 **억지로 벗기지 말 것**
- 화기를 식혀준 뒤 **가위로 조심스럽게 잘라냄**
- 옷이 피부에 붙어 있으면 그대로 두고 주변만 자름

**절대 금지**
- 억지로 옷을 벗기면 피부가 같이 벗겨질 수 있음

#### 3단계: 물집 관리

**물집은 절대 터트리지 말 것**
- 물집은 자연적인 보호막 역할
- 터뜨리면 감염 위험 증가
- 저절로 흡수되거나 터질 때까지 그대로 둠

#### 4단계: 상처 보호

**경미한 화상 (1도)**
- 깨끗한 천으로 가볍게 덮음
- 통기성 있는 거즈 사용

**중등도 이상 화상 (2~3도)**
- 손상 부위를 그대로 드러내거나 깨끗한 천으로 가볍게 덮음
- **즉시 병원으로 이송**

### 절대 하지 말아야 할 것

1. **얼음을 직접 화상 부위에 대지 말 것** - 동상 위험, 조직 손상 악화
2. **물집을 터트리지 말 것** - 감염 위험
3. **화상 부위에 된장, 간장, 소금, 치약 등을 바르지 말 것** - 감염 위험, 치료 방해
4. **알코올로 닦지 말 것** - 피부 손상 악화
5. **기름이나 버터를 바르지 말 것** - 열을 가두어 손상 악화
6. **억지로 옷을 벗기지 말 것**

### 병원 방문이 필요한 경우

#### 즉시 119 신고 및 응급실 방문
- **3도 화상**
- 화상 범위가 넓은 경우 (손바닥 크기 이상)
- 얼굴, 손, 발, 생식기, 주요 관절 부위 화상
- 전기 화상
- 화학 화상
- 호흡곤란 (기도 화상 의심)
- 의식 저하
- 화상의 정도를 구분하기 어려운 경우

#### 병원 방문 권장
- 2도 화상
- 통증이 심한 경우
- 감염 징후 (발열, 고름, 심한 부종, 악취)

### 특수 화상

#### 화학 화상
- 화학물질을 즉시 제거
- 대량의 물로 15~20분 이상 씻어냄
- 화학물질 용기를 병원에 가져가서 성분 확인

#### 전기 화상
- 전원을 차단한 후 접근
- 외상은 작아 보여도 내부 손상이 심할 수 있음
- 즉시 119 신고

## 출혈 응급처치

### 출혈의 종류

#### 1. 동맥 출혈
- 밝은 빨간색
- 박동에 맞춰 분출
- 가장 위험
- 즉각적인 지혈 필요

#### 2. 정맥 출혈
- 어두운 빨간색
- 천천히 지속적으로 흐름
- 동맥 출혈보다 덜 위험하나 지혈 필요

#### 3. 모세혈관 출혈
- 피부 표면에서 스며 나옴
- 가장 경미
- 대부분 저절로 멈춤

### 출혈 응급처치 방법

#### 1단계: 직접 압박 (가장 효과적!)

**방법**
- 상처 범위를 **깨끗한 천이나 거즈**로 덮음
- 손으로 상처 부위를 **직접 세게 압박**
- 최소 5~10분간 지속적으로 압박
- 압박을 풀지 말고 계속 유지

**주의사항**
- 거즈가 피로 젖더라도 떼지 말고 그 위에 새 거즈를 추가로 덮음
- 압박을 자주 풀면 지혈이 어려움

#### 2단계: 붕대 감기

**방법**
- 직접 압박으로 출혈이 줄어들면 붕대로 상처 부위를 감쌈
- 적당한 압력으로 감되 너무 세게 감지 않음

**주의사항**
- 너무 세게 감으면 혈액순환 장애
- 손가락이나 발가락 끝이 차갑거나 파래지면 붕대를 느슨하게 함

#### 3단계: 거상 (높이 올리기)

**방법**
- 손상된 신체 부위를 **심장 부위보다 높게** 올림
- 중력을 이용하여 출혈 부위로 가는 혈류를 감소

**주의사항**
- 골절이 의심되면 함부로 움직이지 말 것

### 상처 종류별 응급처치

#### 1. 찰과상 (긁힌 상처)
**특징**
- 가볍게 입은 상처
- 출혈은 심하지 않으나 감염되기 쉬움

**처치**
- 핀셋이나 족집게로 크고 지저분한 죽은 조직 제거
- 비누와 물로 깨끗이 씻음
- 부종이나 멍을 줄이기 위해 차가운 얼음을 대줌
- 상처가 크거나 옷에 반복되어 쓸리는 부분이라면 항생제 연고를 바르고 붕대로 감음

#### 2. 절상 (베인 상처)
**특징**
- 끝이 예리한 물체(칼, 금속기, 유리파편 등)에 의한 상처
- 감염의 위험은 적으나 **출혈이 비교적 많음**

**처치**
- 직접 압박으로 지혈
- 깨끗한 거즈나 천으로 압박
- 직접 압박으로 지혈이 불가하고 내부 조직이 터져 보일 정도이면 **봉합이 필요** - 병원으로 후송

#### 3. 자상 (찔린 상처)
**특징**
- 뾰족한 물건에 찔린 상처
- 외부 출혈은 적으나 내부 손상이 클 수 있음

**처치**
- 찔린 물체가 그대로 있다면 **절대 뽑지 말 것**
- 물체 주변을 거즈로 고정하여 움직이지 않도록 함
- 즉시 119 신고

#### 4. 절단 (잘린 상처)
**특징**
- 신체 일부가 완전히 또는 부분적으로 절단

**처치**
- 과다 출혈 예방을 위해 압박 붕대로 지혈
- 절단 부위를 심장 높이보다 위로 유지

**절단된 부위 보관 방법** (중요!)
1. 절단 부위를 **생리식염수로 씻어냄**
2. **깨끗한 천으로 싼** 뒤 다시 깨끗한 큰 타월로 두름
3. **비닐봉지에 밀봉**
4. **얼음이 담긴 물통(얼음1:물1)에 넣음**
5. **8시간 이내**면 접합 가능성이 상당히 높음 - 접합 전문병원으로 신속히 이동

**주의사항**
- 절단 부위를 얼음에 직접 닿지 않게 함 (동상 방지)
- 물에 직접 담그지 않음

#### 5. 열상 (찢어진 상처)
**특징**
- 외부 자극에 의해 피부가 찢어진 상처

**처치**
- 흐르는 깨끗한 물로 상처 부위를 닦고 지혈
- 찢어진 상처의 양쪽을 집어 맞추어 봄
- 상처를 꿰매는 것을 고려
- **10분 이상 지혈을 해도 피가 멈추지 않거나**, 환자가 쇼크 증상을 보이면 **즉시 병원 방문**

### 코피 (비출혈)

#### 처치 방법
1. 앉은 자세에서 **고개를 약간 앞으로 숙임** (뒤로 젖히지 말 것!)
2. 코의 무른 부분(콧방울)을 **5~10분간 꾹 눌러 압박**
3. 이마나 코에 찬 수건을 대어 혈관 수축 유도

#### 절대 금지
- 고개를 뒤로 젖히지 말 것 (피가 목으로 넘어가 구토나 질식 유발)
- 코 안에 휴지를 넣지 말 것

### 지혈대 사용 (최후의 수단)

#### 사용 시기
- 심한 동맥 출혈로 생명이 위험한 경우
- 다른 방법으로 지혈이 안 되는 경우
- 사지 절단이나 심한 외상

#### 사용 방법
- 출혈 부위보다 **심장에 가까운 쪽**에 착용
- 넓은 천이나 삼각건 사용 (철사, 끈 사용 금지)
- 혈류가 멈출 정도로 조임
- 지혈대를 건 시간을 기록
- **2시간 이상 유지하지 말 것**

#### 주의사항
- 지혈대는 전문가의 지시 없이 함부로 사용하지 않음
- 잘못 사용하면 사지를 잃을 수 있음

### 쇼크 예방 및 관리

#### 쇼크의 징후
- 피부가 차갑고 창백함
- 식은땀
- 빠르고 약한 맥박
- 빠른 호흡
- 의식 저하
- 불안, 초조

#### 쇼크 예방
- 환자를 눕히고 다리를 약간 올림 (심장으로 혈액 공급)
- 체온 유지 (담요나 옷으로 덮음)
- 의식이 없으면 아무것도 먹이지 말 것
- 지속적으로 환자 상태 관찰

### 병원 방문이 필요한 경우

#### 즉시 119 신고
- 출혈이 심하고 멈추지 않는 경우
- 동맥 출혈
- 절단 상처
- 깊은 자상 (특히 복부, 가슴)
- 쇼크 증상
- 의식 저하

#### 병원 방문 권장
- 10분 이상 압박해도 출혈이 계속되는 경우
- 상처가 깊어 봉합이 필요한 경우
- 감염 징후 (발열, 고름, 심한 통증, 부종, 악취)
- 파상풍 예방접종이 필요한 경우

## 중요 주의사항

### 응급처치 시 기본 원칙
1. 자신의 안전을 먼저 확인
2. 119에 신고
3. 구급대원의 지시에 따라 응급처치
4. 초기 대응을 잘하면 환자의 예후가 좋아짐

### 법적 보호
- 선의의 응급처치자는 법적으로 보호받음
- 고의나 중대한 과실이 없는 경우 민·형사 책임 면제

### 감염 예방
- 가능하면 장갑 착용
- 처치 전후 손씻기
- 깨끗한 재료 사용
